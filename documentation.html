World Tourist Attractions
API Documentation

Documentation

Endpoints

1. Get Tourist Attraction by ID

Endpoint: /api/tourist_attractions/attraction_id

Method: GET

Description: Retrieves details of a specific tourist attraction by its ID.

Parameters:

  • attraction_id (integer): The ID of the tourist attraction.

Response:

  • 200 OK: Returns the details of the tourist attraction.
  • 404 Not Found: If the tourist attraction with the specified ID does not exist.

Example Request:

GET /api/tourist_attractions/1

Example Response:

{
    "id": 1,
    "name": "Eiffel Tower",
    "location": "Paris, France",
    "category": "Landmark",
    "latitude": 48.8584,
    "longitude": 2.2945,
    "image_url": "https://upload.wikimedia.org/wikipedia/commons/thumb/8/85/Tour_Eiffel_Wikimedia_Commons_%28cropped%29.jpg/800px-Tour_Eiffel_Wikimedia_Commons_%28cropped%29.jpg"
}

2. Get Random Famous Location

Endpoint: /api/famous_locations

Method: GET

Description: Retrieves a random famous location from around the world.

Response:

  • 200 OK: Returns the details of a random famous location.

Example Request:

GET /api/famous_locations

Example Response:

{
    "name": "Christ the Redeemer",
    "location": "Rio de Janeiro, Brazil",
    "category": "Landmark",
    "latitude": -22.9519,
    "longitude": -43.2105
}

3. Get Random Tourist Location in India

Endpoint: /api/indias_tourist_locations

Method: GET

Description: Retrieves a random tourist location in India.

Response:

  • 200 OK: Returns the details of a random tourist location in India.

Example Request:

GET /api/indias_tourist_locations

Example Response:

{
    "name": "Red Fort",
    "location": "Delhi, India",
    "category": "Historical Site",
    "latitude": 28.6562,
    "longitude": 77.2410
}

4. Get Wonders of the World Attractions

Endpoint: /api/tourist_attractions

Method: GET

Description: Retrieves a list of wonders of the world attractions.

Response:

  • 200 OK: Returns a random tourist attraction that is considered a wonder of the world.

Example Request:

GET /api/tourist_attractions

Example Response:

{
    "id": 4,
    "name": "Great Wall of China",
    "location": "China",
    "category": "Landmark",
    "latitude": 40.4319,
    "longitude": 116.5704,
    "image_url": "https://upload.wikimedia.org/wikipedia/commons/thumb/2/23/The_Great_Wall_of_China_at_Jinshanling-edit.jpg/1200px-The_Great_Wall_of_China_at_Jinshanling-edit.jpg"
}
